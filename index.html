Logo Usage

On Light Backgrounds

Use the charcoal logo on white or light backgrounds. The charcoal color (#333) provides excellent contrast and readability.

On Dark Backgrounds

On dark backgrounds, invert the logo to white using CSS: filter: brightness(0) invert(1);

Logo Specifications

  • Format: PNG with transparency
  • Logo Files:
    • logo-charcoal-500.png - For light backgrounds (charcoal #333)
    • logo-500.png - Original full color
    • logo--black-500.png - Pure black version
  • Clear Space: Minimum space equal to height of "S" in "Second"
  • Minimum Size: 210px width on mobile, 300px+ on desktop
  • Dark Background Treatment: Use charcoal logo with CSS filter: brightness(0) invert(1)
  • Positioning: Centered on hero/banners, left-aligned in navigation

UI Components

Buttons

Button Specifications

  • Display: inline-block (NOT full width)
  • Border Radius: 0 (sharp corners)
  • Padding: 12px 30px
  • Font: Arial, 16px, normal weight
  • Transition: 0.3s ease on hover
  • Primary (#00BF8F): Darkens to #00a077 on hover
  • Secondary (#b3503e): Darkens to #9a4434 on hover
  • Border: None for filled buttons, 2px for outline buttons
  • Width: Auto (content-based), never full width unless in mobile forms

Form Elements

Form Specifications

  • Border: 2px solid #ccc
  • Border Radius: 0 (sharp corners)
  • Focus State: Border color changes to #00BF8F
  • Padding: 10px
  • Font: Arial, Helvetica, sans-serif, 16px
  • Background: White (#fff) or transparent on dark backgrounds
  • Placeholder Color: #ccc (medium gray)

Image Specifications

Technical Requirements

  • Format: JPEG for photos, PNG for graphics with transparency
  • Resolution: Minimum 1920px wide for hero images, 800px for content images
  • Optimization: Compress images for web (aim for under 500KB per image)
  • Aspect Ratios: 16:9 for hero/banner images, 4:3 or 3:2 for property photos
  • Alt Text: Always include descriptive alt text for accessibility

Overlay Best Practices

  • Dark Overlays: Use rgba(0, 0, 0, 0.4) to rgba(0, 0, 0, 0.8) depending on image brightness
  • Text Contrast: Ensure white text has sufficient contrast over dark overlays
  • Gradient Overlays: Use top-to-bottom gradients (65% to transparent) for subtle effects
  • Hero Sections: 50% overlay is standard for main hero images
  • Search Widgets: 40% overlay works well for form areas
  • Fixed Headers: 80% overlay when scrolled for maximum text legibility

Design Principles

Modern & Clean

Use sharp corners (no border radius), ample whitespace, and clear hierarchy. Let content breathe with generous spacing.

Mountain Aesthetic

Natural color palette inspired by Rocky Mountains. Use landscape photography and earthy tones throughout.

Professional Yet Welcoming

Balance expertise with approachability. Use clean typography and friendly language without sacrificing professionalism.

Image-Forward

Lead with compelling property and landscape photography. Images should be large, high-quality, and showcase the experience.

Mobile-First

Design for mobile devices first, then enhance for larger screens. Ensure touch-friendly targets and readable text.

Functional

Prioritize usability. Clear CTAs, intuitive navigation, and accessible design for all users.

Tone of Voice

We Are:

  • Clear & Direct - Straightforward communication
  • Informative - Provide valuable local knowledge
  • Friendly - Approachable without being casual
  • Confident - Expert-driven but not boastful
  • Helpful - Focus on solving needs

Brand Guidelines Do's & Don'ts

‚úì Do
  • Use the sage green (#00BF8F) as the primary brand color
  • Maintain sharp corners (border-radius: 0) on all UI elements
  • Use high-quality mountain and property photography
  • Maintain logo aspect ratio at all times
  • Use adequate clear space around the logo
  • Invert logo to white on dark backgrounds
  • Apply dark overlays (rgba) for text over images
  • Use generous whitespace and spacing (50px sections)
  • Keep forms clean with 2px borders and #ccc color
  • Emphasize mobile-first responsive design
  • Use Arial (or consider upgrading to Work Sans)
  • Apply #00BF8F color on focus states for forms
  • Keep copy clear, direct, and helpful
  • Showcase outdoor activities and mountain lifestyle
‚úó Don't
  • Don't use rounded corners or border radius
  • Don't distort or stretch the logo
  • Don't change logo colors or add effects
  • Don't rotate or tilt the logo
  • Don't place logo on busy backgrounds without overlay
  • Don't use colors outside the defined palette
  • Don't mix font families (stick to Arial or Work Sans)
  • Don't use heavy font weights for body text
  • Don't use images without proper optimization
  • Don't place text over images without dark overlays
  • Don't use inconsistent spacing or padding
  • Don't create cluttered layouts with insufficient whitespace
  • Don't use jargon or overly technical language
  • Don't forget mobile optimization and testing
