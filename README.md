# Second Home Vacation Rentals - Brand Guidelines

Professional brand guidelines for Second Home Vacation Rentals, a vacation rental property management company in Colorado's Rocky Mountain region.

## Overview

This is a single-page brand guidelines website that documents all visual identity, design principles, and usage standards for the Second Home brand. The entire site is self-contained in one HTML file with all CSS and JavaScript inline.

## Features

- **Complete Brand Documentation**: Colors, typography, logo usage, UI components, spacing, photography guidelines, tone of voice, and more
- **Interactive Elements**:
  - Click color swatches to copy hex codes
  - Click-to-copy markdown export for AI tools
  - Click-to-copy AI image generation prompts
  - Floating navigation menu for easy section jumping
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop
- **Single File**: No external dependencies (except Google Fonts for font comparison)
- **Print-Friendly**: Can be printed for offline reference

## Quick Start

### View Locally

Simply open `index.html` in any modern web browser:

```bash
open index.html
# or
firefox index.html
# or
chrome index.html
```

### Deploy to GitHub Pages

1. **Create a new repository on GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Second Home brand guidelines"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click **Settings** → **Pages**
   - Under "Source", select **Deploy from a branch**
   - Select branch: **main** and folder: **/ (root)**
   - Click **Save**

3. **Access your site**
   - Your site will be live at: `https://YOUR-USERNAME.github.io/YOUR-REPO/`
   - GitHub will provide the exact URL in the Pages settings

## Usage

### For Designers
- Open `index.html` to reference brand colors, typography, spacing, and design principles
- Click color swatches to copy hex codes directly to clipboard
- Use photography guidelines when selecting or creating images
- Follow UI component specifications for buttons, forms, and spacing

### For Developers
- Copy markdown guidelines (at bottom of page) into AI coding assistants
- Reference spacing system and component specs when building new pages
- Use the exact colors and typography documented
- Maintain sharp corners (border-radius: 0) throughout all designs

### For Marketers
- Use AI image generation prompts to create on-brand imagery
- Follow tone of voice guidelines for all copy
- Reference do's and don'ts to maintain brand consistency
- Share guidelines URL with vendors and partners
